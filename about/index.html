Discourse 的一个主要目标是简洁。简洁之下是为帮助社群兴旺发展设计的特性：

100% 开放源代码

放心地将 Discourse 集成到您的站点中 —— 代码属于所有人。

 信任系统

当成员在一段时期内达到要求，他们得到权限，可以帮助维护他们的社群。
移动设备和平板

从第一天起就为移动设计。自适应的移动触屏布局能在您的任何设备上使用。

 社群管理

标记系统让社群控制广告和危险信息，并且让社群和平地解决冲突。
为阅读优化

要阅读更多，只需滚动屏幕。当您到达了主题底部，推荐主题可以供您继续阅读。

 赞

用每个帖子底部的推广按钮 ❤ 表达赞同、支持或者突出有意思的帖子。
概括主题

使用概括按钮压缩长主题，只显示最有趣和最热门的帖子。

 引用上下文

展开引用、展开回复或者展开显示帖子是回复谁的。
国际化

多达17 种语言的支持，还在增长。用户可以在设置面板中自行切换语言。

 实时通知

当有人回复您、引用您的帖子或者提到您的@用户名时，收到私信，指向您帖子的链接，您的帖子被修改或者您的主题被重新分类后您将收到通知。
SEO

友好，人可读的 URL 地址和干净的 HTML。为 Google 索引和搜索优化。

 插件

如果您的站点需要特殊的功能，使用插件支持他们。
单点登录

无缝集成您现在站点的登录系统至 Discourse。

 总在最前的编辑器

在总在最前的编辑中编辑而不需中断阅读 —— 甚至在不同主题浏览时也无需放弃编辑。
第三方服务商登录

轻松添加 Google、Facebook、Twitter、Yahoo、GitHub 和其他常用的社会化登录。

 私密社群

将整个站点设为私有，或者特定的分类。
博客集成

使用我们的 WordPress 插件集成 Discourse 社群至您的博客。或嵌入 Discourse至静态 HTML 站点。

 一键升级

在仪表盘里显示新版本通知。并且可以在网页上一键升级。
频率限制

给所有用户动作设置频率和内容限制；更严格地限制新用户。

 轻松清理垃圾信息

两次点击即可删除垃圾信息，邮件以及 IP 地址封禁，和自动移除所有内容。
封禁垃圾信息

强大的启发式垃圾封禁机制包括新用户沙盒，用户标记封禁以及为链接设置的标准的 nofollow 属性。

 分析

职员使用的仪表盘显示了关于用户、主题以及所有用户行为的重要统计信息 —— 以及最热门来源和主题。
自动 URL 嵌入

像维基百科和 YouTube 这样的流行网站的 URL 将包括一个嵌入的简述。我们支持 OpenGraph 和 oEmbed 标准。

 高级图像处理

通过上传、拖动或者拷贝粘贴添加图片。大图片将自动地生成缩略图和相框照片。
附件

在您的帖子中加入任何允许的格式和大小的文件附件。

 链接追踪

外部链接旁有点击计数器。链入或链接至其他主题的链接将显示在每个主题旁。
投票

快速地创建投票主题。

 剧透警告

标记模糊剧透的内容，点击或者触摸后才会显示。
灵活的文字样式

在帖子中混合使用 Markdown、BBCode 和 HTML 使用样式。

 Emoji 绘文字

先键入 : 然后您就可以使用所有的标准 emoji 绘文字。
丰富的 API

您在屏幕前看到的任何东西，您都可以通过 API 调用获得。Discourse 本身就是一个通过我们的 API 运行的 JavaScript 应用。

 本地头像

内置的本地头像生成，无需外部依赖。另外，选择 Gravatar 或上传自定义头像。
邮件邀请

获得信任等级的用户可以使用邮件邀请朋友加入并点击链接直接回复。管理员可以通过上传邮件地址列表批量邀请用户。

 自动保存草稿

我们自动地在您输入时在后台保存回复和主题草稿至服务器上。
版本历史

追踪每一个编辑的版本，通过一个差异比较器来查看更改。

 维基帖子

让一个帖子可以被有信任等级的社群成员协作更改。
徽章

通过预设的徽章或您自己添加的徽章鼓励积极的社群行为。

 电子邮件通知

当您不在网站上活跃时，您的通知将自动通过邮件发送给您。
邮件回复

无需访问网站即可直接用邮件回复。

 直接更改

在阅读界面直接分割、合并、关闭、存档或者重新分类主题。
标记队列

简单易用的标记队列管理，并可以通过邮件通知。一键同意、不同意或推迟标记。比较历史。

 站点范围通知

本地或者全局固定主题。让一个主题变成所有用户可隐藏的顶部横幅。为紧急情况增加提示。
邮件列表支持

启用特殊模式使所有信息都通过邮件发送给您，就像邮件列表那样。用邮件发表新主题。

 社群中心

官方社群支持中心在 meta.discourse.org，中文论坛位于 meta.discoursecn.org。用于讨论特性、bug 和其他支持请求。
私信

给一个用户或一组用户发送私信。

 网络备份和恢复

在浏览器中备份和恢复您站点的所有数据。只需点击一下，您的数据可以放在任何地方。
自动备份

每天自动备份您的站点，或保存到文件系统，或保存到云。

 CDN 支持

轻松地添加 CDN 提高全球各地的访问速度。
